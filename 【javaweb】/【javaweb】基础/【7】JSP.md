### JSP

--------------------

#### 1. 什么是JSP

示例

```jsp
<%@ page contentType="text/html;charset=UTF-8" language="java"%>
<html>
<body>
<h2>Hello World!</h2>
</body>
</html>

```

- Java Server Page

  java服务器端页面，和servlet一样，用于动态web技术

- 最大的特点

  - 写JSP就像在写HTML，JSP只允许GET、POST、HEAD

  - 与html区别：
    - HTML只给用户提供静态的数据。
    - JSP页面可以嵌入JAVA代码，提供动态数据。

#### 5. 9大内置对象

- 简述

  - pageContext：

  - request：一次性，用户看完就没用了，如：新闻

  - response：

  - session： 一会还有用，本次页面浏览有效，如：购物车

  - application   (ServletContext)：一个用户用完了，其他用户还可以用，如：聊天数据

  - config    (ServletConfig)

  - out

  - page，几乎不用

  - exception

- 存东西，取东西

  ```jsp
  <%
  	pageContext.setAttribute("name1","zyw1");  存东西，在一次页面中有效
  	request.setAttribute("name2","zyw2");      存东西，在一次请求中有效
  	session.setAttribute("name3","zyw3");			 存东西，在一次会话中有效
  	application.setAttribute("name4","zyw4");  存东西，在服务器中有效
  %>
  
  <h1>${name1}</h1>
  <h1>${name2}</h1>
  <h1>${name3}</h1>
  <h1>${name4}</h1>
  
  
  ```

- JSTL标签

  JSTL支持通用的结构化任务，比如迭代，条件判断，XML文档操作，国际化标签，SQL标签。

  除了这些，还提供了一个框架来使用集成JSTL的自定义标签。

  JSTL标签库的使用就是为了弥补HTML标签的不足。

  它自定义了许多标签。可以供我们使用。

  菜鸟教程：https://www.runoob.com/jsp/jsp-jstl.html

  根据JSTL标签所提供的功能，可以将其分为5个类别。

  - **核心标签**

    引入

    ````jsp
    <%taglib prefix="c" uri="http:java.sun.com/jsp/jstl/core"%>
    还有前缀  fmt sql等
    ````

  - **格式化标签**

  - **SQL 标签**

  - **XML 标签**

  - **JSTL 函数**

- 示例

  ```jsp
  <%@ page contentType="text/html;charset=UTF-8" language="java"%>
  <%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
  <html>
  <body>
  <h2>Hello World!</h2>
  
  <h4>if测试</h4>
  
  <hr>
  <form action="coreif.jsp" method="get">
      <input type="text" name="username" value="${param.username}">
      <input type="submit" value="登录">
  </form>
  
  <c:if test="${param.username=='admin'}" var="isAdmin">
      <c:out value="广利元欢迎你">
  </c:if>
  
  <c:out value=${isAdmin}/>
  
  </body>
  </html>
  
  ```
