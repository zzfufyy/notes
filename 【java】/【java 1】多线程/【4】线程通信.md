#### 【4】线程通信

------------------

* 线程通信

  | 方法名             | 作用                                      |
  | ------------------ | ----------------------------------------- |
  | wait()             | 线程一直等待，会释放锁（sleep不会释放锁） |
  | wait(long timeout) | 指定等待毫秒数                            |
  | notify             | 唤醒一个处于等待状态的线程                |
  | notifyAll          | 唤醒同一个对象上所有调用wait()方法的线程  |
