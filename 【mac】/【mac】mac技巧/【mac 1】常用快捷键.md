【mac】常用快捷键

----------

* 切换

  com + tab 切换应用程序

  com + ～   切换窗口

* 通用

  | 键位            | 作用           |
  | --------------- | -------------- |
  | com + h         | 隐藏窗口       |
  | com + ctrl + f  | 最大化缩小窗口 |
  | com + m         | 新建           |
  | com + n         | 新建           |
  | com + o         | 打开           |
  | com + w         | 关闭           |
  | com + q         | 退出           |
  | com + s         | 保存           |
  | com + shift + s | 另存为         |
  | com + space     | 聚焦搜索       |
  
* 截屏

	| 键位                    | 作用         |
	| ----------------------- | ------------ |
	| com + shift + 3         | 截取整个屏幕 |
	| com + shift + 4         | 截取部分屏幕 |
	| com + shift + 4 + space | 截取选定窗口 |
	
* 文件管理器(Finder)

  | 键位              | 作用              |
  | ----------------- | ----------------- |
  | space             | 快速查看          |
  | com + i           | 显示简介          |
  | com + a           | 选择全部          |
  | com + del         | 删除              |
  | com + shift + del | 清空废纸篓        |
  | com + f           | 查找              |
  | com + c           | 复制              |
  | com + v           | 粘贴              |
  | com + opt + c     | 复制当前路径      |
  | com + shift + g   | 输入路径进入/打开 |

* 浏览器

  | 键位    | 作用           |
  | ------- | -------------- |
  | com + + | 放大           |
  | com + - | 缩小           |
  | com + t | 新建选项卡     |
  | com + n | 新建窗口       |
  | com + r | 刷新           |
  | com + w | 关闭当前选项卡 |
