#### 【3】设置注释模板

--------------------------

* Files

  ```java
  /**   
  * @Title: ${file_name} 
  * @Package ${package_name} 
  * @Description: ${todo}
  * @author Zuo Yiwei
  * @date ${date} ${time} 
  * @version V1.0   
  */
  ```

* Type

  ```java
  /** 
  * @ClassName: ${type_name} 
  * @Description: ${todo}
  * @author Zuo Yiwei
  * @date ${date} ${time} 
  */
  ```

* Filed

  ```java
  /** 
  * @Fields ${field} : ${todo}
  */ 
  ```

* Constructor

  ```java
  /** 
  * <p>Title: </p> 
  * <p>Description: </p> 
  */
  ```

* Method

  ```java
  /** 
  * @Description: ${todo}
  * @return ${return_type}
  * @throws 
  */
  ```
