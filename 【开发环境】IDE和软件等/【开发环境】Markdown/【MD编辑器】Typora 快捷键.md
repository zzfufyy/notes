#### 【MD 编辑器】Typora 快捷键

------------------------

* 字体编辑

  加粗：ctrl + b  **OR**  \****

  倾斜：ctrl + i  **OR**  **

  下划线： ctrl + u **OR**  \<u>\</u>

  删除线：alt + shift + 5  **OR**  \~~~~

  上标： \[^abc]  OR \<sup>\</sup>

  下标： \<sub>\</sub>

  高亮： \`abc`

  引用： 顶格 + >asd

  数学公式：顶格 + $$

  插入表格： ctrl + t

  插入链接： ctrl + k **OR** \[TXT](link)

  插入图片： ctrl + shift + i  **OR**  \!\[imgtext](link)

  引用方式：

  ```markdown
  ![text][tag]
  [tag]: link
  
  ```

  插入表情： : + smile + :

  选中单词： ctrl + d

  选中一行： ctrl + l

  选中相同格式： ctrl + e
